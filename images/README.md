# Images Folder

This folder contains images and assets for the WM Attire website.

## Logo
- `logo.png` - Main WM Attire logo (WXA Attire branding)

## Instructions to Add Logo:
1. Save your logo image as `logo.png`
2. Go to GitHub repository
3. Navigate to `images/` folder
4. Click "Add file" > "Upload files"
5. Drag and drop your `logo.png` file
6. Commit the changes
